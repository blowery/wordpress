Displays relative date alongside with your post/comments actual date.

== Description ==

Like 'Today', 'Yesterday', '2 Days Ago', '2 Weeks Ago', '2 'Seconds Ago', '2 Minutes Ago', '2 Hours Ago'.

All the information (general, changelog, installation, upgrade, usage) you need about this plugin can be found here: [WP-RelativeDate Readme](http://lesterchan.net/wordpress/readme/wp-relativedate.html "WP-RelativeDate Readme").
It is the exact same readme.html is included in the zip package.
